PIC32MX_USBOTG_device_firmware

PIC32MX usbotg CDC device open source firmware. Open source PIC32MX usb device framework utilizes full ping-pong buffering capability. As example and possible usage implemented CDC device functionality. In couple with open source TN Kernel RTOS the souftware makes possible to implement high performance and great flexibility embeded device firmware.
